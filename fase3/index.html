Proposito del Sitio

Elaborar recursos educativos digitales para dinamizar las estrategias de aprendizaje, este recurso se explica los temas flex box , listas y enlaces.

¿que es flex box ? Flexbox tiene como meta ser una forma más eficiente de crear diseños, alinear y distribuir espacios entre ítems en un contenedor, incluso cuando las dimensiones de esos ítems son desconocidas y/o dinámicas (en virtud de eso el término "flex").

¿que son listas y enlaces ? nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número. y elance es un inverviculo una opcion mas facil de abrir otra pagina web solo con un clic.
